# vue-project

npm create vue@latest

## Librairies used 

npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
npm i @vueuse/core

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
